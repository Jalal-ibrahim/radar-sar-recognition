## 📝 Project Overview
This project is a high-performance **Deep Learning application** designed to recognize military targets from **Synthetic Aperture Radar (SAR)** imagery. Using the **MSTAR Dataset**, I developed a Convolutional Neural Network (CNN) that can identify armored vehicles with high precision.



> [!TIP]
> **🚀 Live Demo:** [Click Here to Access the App](https://jalal-radar-expert.streamlit.app/)

---

## ✨ Key Features
* 🎯 **Accurate Classification:** Identifies complex military vehicle classes (T-72, BTR-70, 2S1, etc.).
* 🖥️ **Interactive Web UI:** Seamless user experience via a **Streamlit** dashboard.
* 📊 **Real-time Visualization:** Displays the target image alongside the AI's confidence score.
* 🐧 **Optimized for Linux:** Developed and tested for stability on **Linux Mint**.

---

## 📊 Target Classes (MSTAR Dataset)
The AI is trained to distinguish between these strategic military assets:

| Category | Vehicles |
| :--- | :--- |
| **Main Battle Tanks** | T72, T62 |
| **Armored Personnel Carriers** | BTR60, BTR70 |
| **Armored Scout Cars** | BRDM2 |
| **Self-Propelled Artillery** | 2S1 (Howitzers) |
| **Anti-Aircraft** | ZSU-23-4 |
| **Logistics** | ZIL-131 (Military Trucks) |

## 🛠️ Tech Stack
* **AI Engine:** `TensorFlow` & `Keras` (CNN)
* **Frontend:** `Streamlit`
* **Processing:** `NumPy`, `OpenCV`, `Pillow`
* **Hardware:** Developed on a **ThinkPad** running **Linux Mint**
